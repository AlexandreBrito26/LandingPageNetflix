# 🎬 Netflix Landing Page Clone

Bem-vindo(a) ao repositório do **Netflix Landing Page Clone**! Este projeto é uma recriação fiel da página inicial de captura da Netflix, focado em entregar uma interface limpa, moderna e totalmente responsiva.

🔗 **[Acesse o projeto online aqui](http://landing-page-netflix-zeta.vercel.app/)**

## ✨ Funcionalidades

- **Design Responsivo:** A interface se adapta de forma fluida a diferentes tamanhos de tela (mobile, tablet e desktop).
- **Sessões de Apresentação:** Estruturação das vantagens da plataforma (Assista na TV, Downloads para offline, Multi-telas).
- **Planos de Assinatura:** Exibição clara dos planos atuais em formato de cards.
- **FAQ Interativo:** Seção de "Perguntas Frequentes" funcional no formato *accordion* (expansível).
- **Componentes Visuais:** Inputs de e-mail e botões de chamada para ação (CTA) idênticos aos da plataforma oficial.
